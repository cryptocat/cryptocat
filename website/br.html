Converse de forma segura

Cryptocat é um software livre com uma missão simples: permitir que todos possam conversar com seus amigos com privacidade.

  • Open source. Todo o software Cryptocat é publicado de forma transparente.
  • Criptografado por padrão. Cada mensagem é criptografada. Sempre.
  • Conversas seguras. Os chats são seguros, mesmo que suas chaves sejam roubadas.
  • Múltiplos dispositivos. Os dispositivos recebem mensagens mesmo quando estão offline.
  • Compartilhamento de arquivos. Compartilhe arquivos com amigos de forma segura.
Disponível para Windows, distribuições Linux e Mac.
