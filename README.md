### Building
Install dependencies:
```
npm run setup
```

Run:
```
npm run dev
```

### Packaging
Install dependencies:
```
npm install -g electron-builder
```
#### Windows
```
build -w
```
#### Linux
```
build -l
```
#### Mac
```
build -m
```
